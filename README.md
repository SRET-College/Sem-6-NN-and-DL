# Neural Networks and Deep Learning - Semester 6

This repository contains the laboratory implementations for the **Neural Networks and Deep Learning (NN & DL)** course. Each experiment is documented in a Jupyter Notebook, covering topics from basic perceptrons to advanced deep learning architectures like CNNs and RNNs.

---

## 🚀 Experiments Overview

| S.No | File Name | Description |
| :--- | :--- | :--- |
| 1 | `NN_and_DL_Lab_1.ipynb` | Data loading and basic Perceptron implementation. |
| 2 | `NN_and_DL_Expt_2.ipynb` | Implementation of AND, OR, and XOR logic gates with Neural Networks. |
| 3A | `NN_and_DL_Expt_3A.ipynb` | MNIST Digit Classification using Multi-Layer Perceptron (MLP). |
| 3B | `NN_and_DL_Expt_3B.ipynb` | Training an MLP model on the EMNIST Letters dataset. |
| 4A | `NN_and_DL_Expt_4a.ipynb` | MLP with Grid Search hyperparameter tuning for MNIST. |
| 4B | `NN_and_DL_Expt_4b_.ipynb` | Grid and Random Search implementation for Hyperparameter Tuning. |
| 5A | `NN_and_DL_Expt_5a.ipynb` | Implementation of Adaline for logic gates. |
| 5B | `NN_and_DL_5b_.ipynb` | Implementation of Madaline with MR-I learning rule. |
| 6 | `NN_and_DL_Expt_6.ipynb` | CNN for MNIST digit classification. |
| 6B | `NN_and_DL_Expt_6_1.ipynb` | CNN for Intel Image Classification using Kaggle datasets. |
| 7 | `NN_and_DL_Expt_7.ipynb` | Image Classification using CNN and Transfer Learning. |
| 8 | `NN_and_DL_Expt_8.ipynb` | AlexNet, VGG16, and VGG19 for chest X-ray classification. |
